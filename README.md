# fp_datawarehouse
### Implementasi data warehouse terkait OLTP dan OLAP

## SETUP
1. git clone 'URL_REPOSITORY'
2. composer install
3. npm install
4. ubah nama file .env.example menjadi .env
5. php artisan migrate --seed
6. ketik perintah php artisan key:generate


## RUNNING THE APPLICATION
jalankan aplikasi dengan ketik perintah composer run dev
